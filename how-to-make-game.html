CodeInfoWeb Tutorial - Full Steps 1-5

How to make game only use Chrome

Step 1: Open ChatGPT

Open your web browser and go to Google.com. Search for chatgpt.com and click the official link. Log in with your account, or sign up if you don’t have one.

ChatGPT is your coding assistant. Logging in properly ensures you can generate, save, and refine CodeInfoWeb game code efficiently.

Step 2: Enter Your Prompt

Once logged in, type a prompt like this in ChatGPT’s input box (replace [Your Game Name] with your game name): 


Create an HTML game called [Your Game Name] that is visually appealing, not plain, and includes animations.

Example:


Create an HTML game called CodeInfoWeb that is visually appealing, not plain, and includes animations.
Being specific ensures ChatGPT generates interactive, dynamic code. Copy all output including <html>, <style>, and <script> tags. Refine code if needed.

Step 3: Paste Code into CodeInfoWeb Online Editor

  1. Go to CodeInfoWeb.
  2. Click with you email or google account.
  3. Click My project and click Create new project.
  4. Delete any default code in the editor.
  5. Paste the code from ChatGPT.
  6. Click Run or use preview to test:
    • Animations
    • Buttons & controls
    • Audio & images
  7. Click the Export button "
Using the editor lets you debug instantly. Testing here first ensures your CodTor game works before saving locally.
  • Test locally by opening index.html to verify animations, audio, and images.
  • Keep a backup folder.
  • Folder & file names are flexible; repository name in GitHub will follow what you choose.
    • Proper organization ensures smooth uploading to GitHub. Relative paths must match folder structure.

    Step 4: Publish CodeInfoWeb on GitHub Pages

    1. Go to GitHub and sign up or log in.
    2. Create a new repository:
      • Click + → New repository
      • Name it anything you like (e.g., CodeInfoWeb, MyHTMLGame)
      • Set visibility to Public
      • Check Add a README
      • Click Create repository
    3. Upload your game files:
      • Click Add file → Upload files
      • Select HTML (or ZIP) file
      • Click Commit changes
    4. Enable GitHub Pages:
      • Go to Settings → Pages
      • change from None → main
      • Click Save and refresh several times for link appears
    5. Access & share your game:
      • Copy the GitHub Pages URL
      • Open a new tab and paste the URL, append your HTML filename
      • Example: https://username.github.io/YourRepoName/CodeInfoWeb.html
    Your CodeInfoWeb game is now live online! Test on multiple devices to ensure all assets load correctly. Keep local backups and update files via GitHub commits when needed.
